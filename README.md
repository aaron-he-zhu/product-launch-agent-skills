# Product Launch Agent Skills (RAMP)

> [!IMPORTANT]
> **This is a signpost repo — the skills live in [aaron-marketing-skills](https://github.com/aaron-he-zhu/aaron-marketing-skills).**
> The 16 skills listed below are actively developed there, as one of 5 disciplines in an 86-skill bundle sharing one contract, six benchmark-driven gates, and keyless data connectors. This repo carries no skill content — it exists so you can find the discipline; the bundle is where you install it.

16 skills covering the RAMP loop — **Research → Assemble → Mobilize → Prove** — scored by the [RAMP benchmark](https://github.com/aaron-he-zhu/launch-marketing-ramp-benchmark), with `launch-readiness-auditor` as the LQS pre-launch gate (T-1 go/no-go). Entrypoint command: `/aaron-marketing:launch`.

## Install (from the umbrella bundle)

| Host | Install |
|------|--------|
| **Claude Code** | \`/plugin marketplace add aaron-he-zhu/aaron-marketing-skills\` then \`/plugin install aaron-marketing@aaron\` |
| **Codex · Cursor · OpenCode · Antigravity · Gemini CLI · Copilot CLI · OpenClaw · Hermes · [70+ hosts](https://github.com/vercel-labs/skills#supported-agents)** | \`npx skills add aaron-he-zhu/aaron-marketing-skills\` |
| **Single skill** | \`npx skills add aaron-he-zhu/aaron-marketing-skills -s launch-readiness-auditor\` |
